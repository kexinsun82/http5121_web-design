Intro to flexbox

Flexbox is a layout system in CSS. It consists of a wrapper container called a "flex container". The children of this flex container are the elements you want to "lay out". They are called "flex items".

Different properties are used for the flex container and for the flex items. For more predictable behaviour, do not set flex item properties on a flex container and vice versa.

As an example the below example is using divs just to demo the properties but you can use any element as the flex container (e.g. section) and any element as a flex item (e.g. if building a page layout, main could be a flex item).

In most cases, horizontal layouts are the most complicated and you'd want the vertical layout to grow according to content length so I use the row layout in the majority of cases.
